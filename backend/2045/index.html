Join numbers to get to the 2048 tile!

How to Play

  • Use your arrow keys or swipe on mobile to move the tiles.
  • When two tiles with the same number touch, they merge into one!
  • After each move, a new tile (either 2 or 4) will appear in a random empty spot.
  • Your goal is to create a tile with the number 2048.
  • The game ends when there are no more valid moves available.
