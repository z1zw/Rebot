### 2.2 时空闭包与傅里叶分析 (spatiotemporal_closure.py - 874行)

**理论基础** (学术级创新):
```
协作信号的傅里叶表示:
ĝ(ξ) = F{g}(ξ) = ∫ g(t) e^{-2πiξt} dt

节点交互张量:
X ∈ ℝ^{|V|×|V|×T}
X[i,j,t] = 节点i到j在时刻t的交互强度

L²闭包等价关系:
g₁ ~ g₂ ⟺ ‖g₁ - g₂‖_{L²} = 0
```
